# Silvermont: Fragments v0.9.0

Text-based HTML otome/life-sim vertical slice.

- Week 1 starts Monday, September 2; September 1 remains the prologue.
- Seven independent Monday–Sunday planner selections.
- Calendar-style planner UI.
- Previous week copies all seven selections forward.
- Events now also use activity tags; major plot events use `any`.
- Existing weekday/weekend tags and weekly caps remain.
